\section Type File 
Terdapat tiga tipe file di sistem operasi yaitu tipe file:
\begin{enumerate}
\item Reguler
	Berisi Informasi, terdiri dari file ASCII (berisi teks) dan biner (untuk dieksekusi misalnya file .EXE)
\item Directory
	Dimiliki system untuk mengelola struktur system file, dan merupakan file yang berisi informasi mengenai file-file yang termasuk dalam direktori
\item Spesial
	Merupakan nama logik perangkat I/O -> dapat dipandang sebagai file, untuk mempermudah pemakai. 
\end{enumerate}
\begin{enumerate}
\item File spesial karakter
	Berhubungan dengan perangkat I/O -> terminal, printer, port jaringan, Modem .
\item File spesial blok
	Berhubungan dengan perangkat I/O sebagai kumpulan blok data -> penyimpanan sekunder
\end{enumerate}
